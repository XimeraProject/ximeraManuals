\section{Videos}

We can embed YouTube Videos using the syntax \verb|\youtube{ID}|, where ID
is the video ID found at the end of the YouTube link after ``watch?v=''.
\begin{verbatim}
\begin{center}
\youtube{FvgF95i0_lw}
\end{center}
\end{verbatim}
which would embed the video into the page, like this:
\begin{center}
  \youtube{FvgF95i0_lw}
\end{center}
